## 📁 ディレクトリ構造
```
.
├── 00_templates
│   ├── diary.md
│   ├── note.md
│   └── task.md
├── 01_diary
├── 02_notes
├── 03_tasks
├── 04_scripts
├── 05_temp
├── 99_data-view
├── MyKanban
└── README.md
```

### 00_templates
テンプレートを保管するディレクトリ、テンプレートを追加したくなったらここに追加

### 01_diary
デイリーノートが作成される場所、`01_diary/{year}/{month}` 配下に作成される

### 02_notes
新規ノートはここに作成される
デイリーノートと同じような仕組みで作成される

### 03_tasks
タスクが作成される場所、なんかチケット単位で作っていくのがイメージしやすいと噂

### 04_scripts
なんか便利なスクリプトを作成したらここにおきたいカモってところ、未定

### 05_temp
ほんとに一時的なメモで使うところ、最悪使わないかも

### 99_data-view
作成したノートのデータビューをするところで使う予定。まだ使っていない
