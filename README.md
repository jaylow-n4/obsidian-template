## 📁 ディレクトリ構造
```
.
├── 00_templates
│   ├── diary.md
│   ├── note.md
│   └── wikis.md
├── 01_diary
├── 02_notes
├── 03_wikis
├── 04_assets
├── 99_data-view
├── MY TODOs
└── README.md
```

### 00_templates
テンプレートを保管するディレクトリ、テンプレートを追加したくなったらここに追加

### 01_diary
デイリーノートが作成される場所、`01_diary/{year}/{month}` 配下に作成される

### 02_notes
新規ノートはここに作成される
デイリーノートと同じような仕組みで作成される
タスクごとに作成するようなイメージ

### 03_wikis
知識を格納するところ

### 04_assets
画像などのファイルを格納する

### 99_data-view
作成したノートのデータビューをするところで使う予定。まだ使っていない
